## Scope

This pass reviewed `DOD_111689011.mp4`, the hard DVIDS `DOW-UAP-PR34` / `DoW-UAP-D33` pair, from `4.0s` through `59.0s`. This interval covers the DVIDS entry, horizontal back-and-forth, and generally-centered phases before the `01:00` blue-reticle designation phase begins.

The intent was to test whether the public MP4 supports a structured image-plane maneuver lane. It was not intended to validate the report's `80 MPH` value.

## Method

Generated by `https://github.com/dfredriksen/ufo-uap-release-corpus-assessment/blob/main/scripts/ufo_pr34_d33_manual_track_pass.py`.

## Caveats

- These events are not independent proof of real-world 90-degree turns.
- The sensor is panning and tracking, so image-plane direction changes are not the same as object-world trajectory changes.
- The public MP4 lacks FOV/zoom state, slant range, platform motion, gimbal pointing, target coordinates, and frame-level telemetry.
- The report's `80 MPH` value remains report-derived.
- The interpolated rows should be used only for continuity visualization and path-shape review; they should not be treated as visually detected object centers.

## Updated Judgment

PR34/D33 is now stronger than a simple release/report match. It has a bounded pre-reticle image-plane track with multiple sharp apparent heading changes. The correct conclusion is narrow: the MP4 supports a maneuver-like image-plane pattern, but it still cannot validate physical turns or speed without sensor geometry.

## Follow-On Geometry Pass

Dedicated note: `https://github.com/dfredriksen/ufo-uap-release-corpus-assessment/blob/main/research/ufo-pr34-d33-geometry-feasibility.md`

The follow-on feasibility pass found that if the cleaned path-average image-plane rate of `118.837 px/s` represented an `80 MPH` target, the implied slant range would be about `8.94 nm` at `2 deg` full-frame HFOV, `4.47 nm` at `4 deg`, or `2.23 nm` at `8 deg`.

This does not validate the report speed. It shows that the reported speed is not falsified by the MP4 under some simple FOV/range scenarios, while still being underdetermined without sensor geometry.
