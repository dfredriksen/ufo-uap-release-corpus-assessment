## Scope

This pass tests the `DoW-UAP-D33` report value of `80 MPH` against the bounded `DOD_111689011.mp4` manual-review track from `4.0s-59.0s`.

The purpose is not to prove the reported speed. The purpose is to quantify what range/FOV combinations would be required if the public MP4 image-plane rates were treated as angular motion.

This shows the core limitation: the same observed image-plane rate can be compatible with very different real-world speeds depending on assumed range and FOV.

## Turn-Radius Scenarios

If the seven image-plane heading-change events were physical turns at `80 MPH`, the simple implied turn radii would range from about `36.5 m` to `100.0 m`.

That result should be treated as a warning label rather than a conclusion. The heading changes are image-plane heading changes from a panning/tracking sensor, not confirmed object-world turn angles.

## Judgment

The PR34/D33 public MP4 does not falsify the report's `80 MPH` value. It also does not independently validate it.

The strongest defensible statement is:

`DOD_111689011.mp4` supports multiple sharp apparent image-plane heading changes during the D33 pre-reticle interval. If the path-average image-plane rate represented an `80 MPH` target, the implied slant range would be on the order of `8.94 nm` at `2 deg` full-frame HFOV, `4.47 nm` at `4 deg`, or `2.23 nm` at `8 deg`. Without actual sensor geometry, these remain illustrative scenarios only.
