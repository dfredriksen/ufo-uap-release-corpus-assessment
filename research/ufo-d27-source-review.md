## Bottom Line

`D27` is one of the strongest report-text cases in the current corpus. The report preserves a precise UAP time, event serial, aircraft context, target-pod / FMV context, over-water movement language, a distinctive object description, a precise first-accuracy field, and useful negative controls. The key UAP description is a glowing hot spherical unidentified object with a vertical unwavering cylindrical pole/bar attached on the bottom, with the caveat that it may instead be a reflection from the object in water.

`D27` is also the best written-report content match for DVIDS `DOW-UAP-PR29` / local `DOD_111688964.mp4`. DVIDS labels the accompanying mission report as `DoW-UAP-D8`, but the DVIDS narrative matches D27's pole/bar/reflection language and conflicts with the official/local D8 two-white-hot-object report. Treat `DOD_111688964.mp4` as hard PR29 media, D27 as the written-report content match, and the DVIDS `D8` label as an unresolved release-index discrepancy.

The main analytic limit remains kinematic: the public MP4 and report do not provide source FOV, range, platform motion, gimbal pointing, or raw telemetry. The `140 KNOTS` UAP speed is report-derived, not independently validated from public video.

## Description And Video Alignment

The released report description says the object was glowing hot and spherical, with a vertical unwavering cylindrical pole/bar attached to the bottom, and adds that this may have been a reflection from the object in the water. DVIDS `PR29` describes the public video as a 21-second infrared clip in which an area of contrast resembles an inverted teardrop with a vertically linear trailing mass. That is a close content match to D27, even though the DVIDS page labels the accompanying report as D8.

Prior local PR29 review already found:

- `DOD_111688964.mp4` is the hard DVIDS PR29 media file.
- The clip shows one persistent bright object/contrast feature over water-like texture.
- Dense marking found a repeated short downward/vertical bright feature, making the D27 pole/bar/reflection description visually plausible.
- The public clip does not prove a physical appendage, reflection, exact event time, UAP speed, or UAP altitude.

## Analytic Treatment

- Confidence is high for D27 as the PR29 written-report content match.
- Confidence is high for the event time, event serial as printed, report context, first-coordinate lane, UAP speed field, and negative controls as report text.
- Confidence is medium for the pole/bar/reflection description: it is preserved in the report and visually plausible in PR29, but still not resolved as physical structure versus reflection or sensor/video artifact.
- Confidence is low for independent kinematics. `140 KNOTS` is an estimated report value, and the public MP4 lacks FOV, range, platform motion, gimbal pointing, and telemetry.
- Do not treat the friendly aircraft fields (`23,999FT`, `163 KNOTS`, trajectory `294`) as UAP altitude/speed/trajectory.
