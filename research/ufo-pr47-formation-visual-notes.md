## Findings

PR47 is now the strongest standalone multi-object/formation visual lane in the local corpus. The full pass found three-component groups in `349/598` sampled rows and high-or-medium support in `466/598` sampled rows.

The relative spacing is compact and broadly stable at image-plane scale. Median max pair distance stays close across phases:

- `48.430 px` in the upper-field formation track
- `51.264 px` in the upper-to-mid descent
- `51.080 px` in the reticle-near pass
- `48.911 px` in the late lower-field interval

The detected cluster persists through field movement. It starts high in the crop, drifts through the upper-to-mid descent interval, becomes more confounded near the reticle, and remains detectable again in the late lower-field interval.

The main caveat is overlay/reticle confounding. Some rows intersect or sit near colored reticle/overlay pixels, especially as the cluster approaches the centerline. The detector is useful for audit support, but rows should not be treated as clean physical object centroids.

## Interpretation

The full PR47 pass supports the public-release description of three contrast areas maintaining fixed relative positions. It does not establish whether those areas are three independent objects, a single structured object with multiple hot/bright regions, a sensor/processing artifact, or another conventional target presentation.

Do not use PR47 for physical kinematics. The public MP4 does not expose range, FOV, platform motion, zoom state, target coordinates, altitude, object size, object separation, or speed.
