## Source Handling

This is a release-index completeness result, not a physical-validation result. A hard DVIDS page identity proves that a local MP4 maps to a public release page. It does not validate speed, range, altitude, maneuvering, object identity, number of independent objects, or sensor geometry. Those still require source telemetry, FOV/zoom state, slant range, platform motion, and target coordinates.
