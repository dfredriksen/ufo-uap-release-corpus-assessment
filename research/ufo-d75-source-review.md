## Bottom Line

`D75` is a useful Gulf of Aden mission-report case, but the review downgrades its descriptive value. The source PDF confirms the free-text UAP description is not recoverable from OCR: the visible line stops at `UAP HAD`, and the following `Gentext` box is redacted. The structured UAP fields are therefore the reliable evidence lane.

The strongest preserved facts are: one UAP observed at `140517:00ZJUL24`, FMV as the primary sensor in the mission section, first/last MGRS-style coordinates in `38P MT`, estimated low altitude, estimated northwest trajectory, no observer engagement, no effects on persons or equipment, no recovered material, and the note that the observer followed the UAP until distance made it too far to follow. The report also states the UAP's speed was faster than the observing platform's flying speed, but the actual velocity value is redacted.

Do not use D75 for object-shape or precise-kinematics claims. Treat it as a structured, document-only Gulf of Aden observation with redacted descriptive fields.

- `UAP Description ...: UAP HAD`
- `Gentext (UAP Event Description):` followed by a redacted classification marker and a large `1.4a` box.

This means the earlier "truncated description" is not recoverable by better OCR from the released PDF. The releasable source does not preserve the object's visible shape, color, markings, or detailed narrative beyond the structured fields.

## Analytic Treatment

- Confidence is high for the event time and general Gulf of Aden / `38P MT` lane.
- Confidence is medium for movement: straight path, same altitude, low altitude, and NW trajectory are preserved, but still estimated.
- Confidence is low for physical speed because the value is redacted; only the relative statement "faster than the observing platform's flying speed" is visible.
- Confidence is low for object description because the description field is visibly truncated/redacted.
- No local or public PR/video pairing has been found.

## Follow-Up

1. Keep D75 as a high-priority document-only Gulf of Aden report.
2. Stop treating the object description as an OCR problem; the released source redacts the useful narrative.
3. Search only for official video/source metadata that names `D75`, MISREP `10194673`, `140517ZJUL24`, or the `38P MT` coordinate lane.
4. If source video appears, prioritize the interval around `140517ZJUL24` and compare first/last coordinate movement against the `NW` trajectory field.
