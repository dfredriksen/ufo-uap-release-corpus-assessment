# Review Packets

This folder contains two independent review packets:

- `chatgpt-pro-touchup-packet.md` asks ChatGPT Pro to improve the paper's professional polish without strengthening unsupported claims.
- `chatgpt-pro-polished-paper.md` is the polished draft returned from ChatGPT Pro.
- `claude-review-packet.md` asks Claude to critique reasoning, evidence hierarchy, clarity, and overclaim risks.

The intended workflow is:

1. Send the publication paper and relevant support links to ChatGPT Pro for editorial refinement.
2. Send the paper and support links to Claude for critical review.
3. Capture returned comments or proposed revisions in this folder before changing `paper.md`.
4. Accept only changes that preserve the paper's evidence boundaries.

This process is included in the paper's method section as a publication-stage quality-control step.
